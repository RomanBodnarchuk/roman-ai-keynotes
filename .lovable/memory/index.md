# Project Memory

## Core
Premium AI keynote speaker website for Roman Bodnarchuk. Dark luxury aesthetic, Apple-level polish.
Bebas Neue headings, DM Sans body, DM Serif Display for accent titles. No em dashes. All billing in USD.
Primary color: red (#c8102e / HSL 352 85% 43%). Gold accent for pricing. Black/graphite backgrounds.
Never discount. Premium positioning is the brand.
Roman is solo founder, zero employees, all AI agents. Based in Toronto.
Border-radius: 0 (sharp edges). No rounded corners on cards/buttons.
