Typography: Bebas Neue (headings), DM Sans (body), DM Serif Display (accent/topic titles)
Primary: HSL 352 85% 43% (red #c8102e)
Gold accent: HSL 42 55% 54% for pricing text
Background: HSL 0 0% 4% (near-black)
Card: HSL 0 0% 10% (dark graphite)
Border-radius: 0 everywhere. Sharp, masculine edges.
Buttons: No border-radius. Uppercase tracking-[2px].
Layout: Full-bleed sections with 2px gaps between cards (grid gap-[2px]).
Stats bar: Full-width red background with white text.
Authority strip: Ventures listed with Bebas Neue, red accent on suffixes.
Topic cards: Hover reveals red top-border via scaleX animation.
Testimonials: DM Serif Display italic quotes with red bar accent.
